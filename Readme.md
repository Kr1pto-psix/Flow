# Flow for windows
Flow программа для контроля времени, помогает повысить вашу продуктивность, евляется полной копией Flow из Mac os

## Функционал

- [ ] Блокировка url
- [ ] Отоброжение времени в трее
- [ ] Минималистический дизайн
- [ ] Мотивационые фразы
- [ ] Два дизайна (dark and light)
